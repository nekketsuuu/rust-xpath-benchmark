# fixtures

This directory contains XML files used as benchmark inputs.

## Artificial data

The following files are generated by `fixtures/generate/` and can be
regenerated at any time by running from the repository root:

```
cargo run -p generate
```

| File | Content | Size |
|---|---|---|
| `small.xml` | 10 books | ~2 KB |
| `medium.xml` | 1,000 books | ~160 KB |
| `large.xml` | 50,000 books | ~8 MB |
| `deep-nest.xml` | 200 levels of nesting with one book at the leaf | ~85 KB |
| `wide.xml` | 100,000 sibling book elements (flat) | ~10 MB |

All generated files share the same bookstore schema:

```xml
<bookstore>
  <book category="fiction">
    <title lang="en">…</title>
    <author>…</author>
    <year>…</year>
    <price>…</price>
  </book>
  …
</bookstore>
```

The generator uses a fixed random seed by default (`DEFAULT_SEED_BOOKSTORE = 42`,
`DEFAULT_SEED_WIDE = 123`) so repeated runs produce identical output.
To change the seed, edit the constants in `fixtures/generate/src/main.rs`.
